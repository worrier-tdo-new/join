1. PROJECT OVERVIEW
--------------------------------------------------
EduSphere is a modern, responsive educational website 
template built with pure HTML, CSS, and vanilla JS.
It is styled with a scholarly theme (royal blue + golden 
highlight) and is fully responsive across desktops, 
tablets, and mobile devices.

Main Features:
- Responsive navigation with mobile hamburger toggle
- Hero section with call-to-action
- Course, Live Classes, Exams, Results, Blog pages
- User login & signup forms
- Policy pages (Privacy, Terms, Sitemap)
- Modal for "More" navigation items
- Dark academic theme with professional typography

3. RESPONSIVENESS
--------------------------------------------------
- Breakpoints at 1280px, 980px, 720px, and 520px.
- At <=980px: 
  • Navigation collapses into a hamburger menu.
  • Layout grids (hero, features, stats, blog) stack.
- At <=720px: Buttons shrink, forms become single-column.
- At <=520px: Buttons expand full width for easier taps.

--------------------------------------------------
4. HOW TO USE
--------------------------------------------------
1. Open index.html in a browser to preview the site.
2. Update text content and replace placeholder images 
   in /assets/img with your own.
3. Modify /assets/css/style.css to tweak colors/fonts.
4. Add or edit navigation links in each page’s header.
5. Use /pages directory for creating new content pages.

--------------------------------------------------
5. JAVASCRIPT FUNCTIONS
--------------------------------------------------
- main.js handles:
  • Mobile nav toggle (hamburger menu)
  • Modal toggle for "More" navigation
  • Closing nav menu on link click (mobile)

--------------------------------------------------
6. CUSTOMIZATION
--------------------------------------------------
- Colors and typography can be customized in :root 
  section of style.css.
- Replace logo.svg with your institution’s logo.
- Update sample blog post or add more posts in /posts.
